Node와 Element의 차이

        Node : 태그(요소 노드), 속성, 주석, 내용(텍스트 노드) 모두 표현
        Element : Node 의 하위 개념으로 요소 노드만을 표현

        [Element만 탐색하는 방법]
        children : 자식 요소만 모두 선택
        parentElement : 부모 요소 선택
